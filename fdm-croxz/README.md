# Croxz - FDM Universal Downloader

A Free Download Manager 6 add-on for downloading media from YouTube, Twitter, Instagram, and 50+ other sites using yt-dlp.

## Features

- Video downloads with embedded audio (no separate audio-only files)
- Quality selection (4K, 1080p, 720p, etc.)
- Audio-only option for music downloads
- File size displayed for each quality option

## Requirements

- [Free Download Manager 6](https://www.freedownloadmanager.org/)
- [yt-dlp](https://github.com/yt-dlp/yt-dlp) (installed and in PATH)
- Python 3.8+

## Installation

1. Download `fdm-croxz.fda` from Releases
2. Open FDM > Tools > Add-ons
3. Click "Install from file"
4. Select the `.fda` file
5. Restart FDM

## Usage

Paste a video URL into FDM. The add-on automatically extracts available formats with quality options.
